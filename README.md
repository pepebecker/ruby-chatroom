# Ruby Chatroom

[![MIT-licensed](https://img.shields.io/github/license/pepebecker/ruby-chatroom.svg)](https://opensource.org/licenses/MIT)
[![chat on gitter](https://badges.gitter.im/pepebecker.svg)](https://gitter.im/pepebecker)

## Usage

#### Start the Server
```bash
ruby chatroom.rb -p <port>
```

#### Connect to the Server
```bash
telnet 127.0.0.1 <port>
```
